Features

Spreadsheet feel, editor scope.

Excel-shaped

Sticky header + row numbers

Column letters across the top, row numbers down the side, both pinned. The grid you already know.

Virtualized

Million-row CSVs without freezing

Only the visible window is in the DOM. Scroll, the window slides, nodes recycle. No janky frame drops.

RFC-4180

Round-trip faithful

Quoted fields, embedded commas, embedded newlines, double-quote escapes — all preserved on save.

Blank scratch sheet

Boot into a writable surface

50 × 10 empty cells, no file path. Start typing immediately and Save As to give it a home.

Per-session undo

Ctrl+Z while editing

In-memory undo stack across cell edits within a session. Save clears it.

Tauri · Rust · csv crate

Native Linux

Small binary, AppImage and .deb, real .csv file association, in-app updater. No Electron.

Principles

What this isn't.

No formulas. This isn't a calculator. It's a tabular text editor that happens to look like Excel.
No multi-sheet workbooks. A CSV file is one table. If you need tabs, you don't want a CSV editor.
No charts, pivots, conditional formatting. If you need analysis, the CSV is your input — open it somewhere else.
No dialect detect (v1). Comma only. Semicolon and TSV come in v2 when we add a raw-text mode.
Install

Two files. One minute.

Both formats ship the same binary and register .csv as an associated file type — opens land in this app by default.

Or build from source: pnpm install && pnpm tauri build.

AppImage · any Linux

# download
curl -LO https://github.com/krill-software/csv-editor/releases/download/v0.2.1/CSV.Editor_0.2.1_amd64.AppImage

# make executable & run
chmod +x CSV.Editor_0.2.1_amd64.AppImage
./CSV.Editor_0.2.1_amd64.AppImage orders.csv

Debian / Ubuntu

sudo dpkg -i CSV.Editor_0.2.1_amd64.deb
krill-csv-editor orders.csv
